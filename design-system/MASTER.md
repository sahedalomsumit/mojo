# Design System: Mojo 3D Landing Page

## Brand Identity
- **Primary Color:** `#FF0000` (Mojo Red)
- **Secondary Color:** `#000000` (Deep Black)
- **Accent Color:** `#FFFFFF` (Pure White)
- **Neutral:** `#1A1A1A` (Dark Gray)

## Typography
- **Heading:** 'Teko', sans-serif (via Google Fonts)
- **Body:** 'Inter', sans-serif (via Google Fonts)

## Component Tokens
- **Buttons:**
  - Background: `#F40009`
  - Text: `#FFFFFF`
  - Hover: Brightness increase + scale
- **Cards:**
  - Background: `rgba(255, 255, 255, 0.05)`
  - Border: `rgba(255, 255, 255, 0.1)`
  - Backdrop Filter: `blur(10px)`

## 3D Scene Config
- **Ambient Light:** Intensity 0.5
- **Spotlight:** Intensity 2, Position [5, 5, 5]
- **Environment:** Studio HDRI (simulated via Drei)
- **Camera:** FOV 45, Position [0, 0, 5]
